# node-mini-auth

## JWT auth example using cookies with express, mongo.

## 🚧 How to start-stop dev environment

### 👍 Start

```bash
npm run compose:dev:up
```

### 👎 Stop

```bash
npm run compose:dev:down
```

## Endpoints

### Login

```bash
/auth/login
```

### Register

```bash
/auth/register
```
